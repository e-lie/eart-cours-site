## 在Mac Os上

用搜索或在系统实用程序中打开 "Terminal"。复制并逐一运行以下命令。

- `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- `brew install python`
- `brew install thonny`
- `pip3 install pillow`
- `brew install --cask krita`
